SYSTEM REQUIREMENTS:
--------------------
• Python 3.8 or higher
• Operating System: Windows, Linux, or macOS

WHAT GETS CREATED DURING SETUP:
--------------------------------
• .venv/ folder                - Virtual environment (optional but recommended)
                                Size: ~200-500 MB
                                Contains isolated Python packages

• __pycache__/ folders         - Python bytecode cache
                                Created automatically by Python
                                Safe to delete (will be regenerated)

• *.pyc files                  - Compiled Python files
                                Created automatically
                                Safe to delete (will be regenerated)

MIGRATION TO NEW SYSTEM:
------------------------
To move this dashboard to another computer:
1. Copy the ENTIRE Portable-Dashboard-Deploy folder
2. On new system, run setup.bat (Windows) or setup.sh (Linux/Mac)
3. Run start_dashboard.bat (Windows) or start_dashboard.sh (Linux/Mac)
4. Done! All your data comes with you.


VERSION CONTROL:
----------------
If using git or other version control:
- DO commit: All application code and scripts
- DO commit: requirements.txt
- CONSIDER committing: db.sqlite3 (for dev), or exclude for prod
- DO NOT commit: .venv/ folder
- DO NOT commit: __pycache__/ folders
- DO NOT commit: *.pyc files
