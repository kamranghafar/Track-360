Import Instructions
File Format

The import file should be in CSV or Excel format with the following columns:

  • name (required): The name of the resource
  • email: The email address of the resource
  • role: The role of the resource
  • skill: The skill of the resource (automation, manual, or both)
  • availability: The availability of the resource (true/false, yes/no, 1/0)
Notes
  • If a resource with the same name already exists, it will be updated with the new information.
  • If a column is missing or empty, the corresponding field will be left blank or set to its default value.
  • The skill field must be one of: automation, manual, or both.
  • The availability field should be a boolean value (true/false, yes/no, 1/0).
