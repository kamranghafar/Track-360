Status Workflow

The typical SOP status workflow is:

  1. In Progress - SOP is being drafted or updated
  2. Under Review - SOP is being reviewed by stakeholders
  3. Review Done - Review is complete, ready for rollout
  4. Rollout in Progress - SOP is being implemented
  5. Rollout Done - SOP has been fully implemented
  6. Active - SOP is currently in use
  7. Inactive - SOP is no longer in use

Note: Status changes are logged in the SOP's history.
